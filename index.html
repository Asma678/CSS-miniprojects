What is CSS?

  1. CSS stands for Cascading Style sheets

  2. CSS description how html elements are to be display on screen,paper or in other media

  3. CSS is not necessry CSS saves a lot of work. It can Control the layout of multiple web pages at all

  4. External stylesheets are stored in CSS file

CSS can be added in HTML documents in three way:

  • Inline- by using the style attribute inside HTML elements
  • Internal- by using inline element in the head section
  • External- by using link element to link to an external CSS file
  • The most common way to add CSS, is to keep the style in external CSS file. However, in this tutorial we will use inline and internal styles,because this is easier to demonstrate, and easier
    for you to try it yourself

when comparing CSS class vs ID, the difference is that CSS class applies a style to multiple element. ID, on the other hand, applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it's used by javascript

In CSS, selectors are used to target a specific element or range of elements on a web page. once an elements has been targeted, a style or set of styles can be applied to the element.

There is a wide range of Selectors available. Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
